1. Learn the purpose of <form> in HTML. Create a basic form structure with action and method.
<form> is used to collect user input.
It sends data to a server for processing.
Common uses:
Login pages
Signup/registration forms
Contact forms

<form action="login.php" method="POST">
</form>
2. Practice using <input type="text">, <input type="password">, and <input type="email">. Create a login form.

3. Use checkboxes for multiple choices, radio buttons for single choice, and number fields for numeric input.

Checkboxes 
Used when the user can select more than one option
Each option is independent
Example:
<label>Hobbies:</label><br>
<input type="checkbox" name="hobbies" value="reading"> Reading<br>
<input type="checkbox" name="hobbies" value="music"> Music<br>
<input type="checkbox" name="hobbies" value="sports"> Sports<br>

Radio Buttons 
Used when the user must select only one option
All options must have the same name
Example:
<label>Gender:</label><br>
<input type="radio" name="gender" value="male"> Male<br>
<input type="radio" name="gender" value="female"> Female<br>
<input type="radio" name="gender" value="other"> Other<br>

Number Input 
Used to accept only numbers
Can restrict range using min and max
Example:
<label for="age">Age:</label>
<input type="number" id="age" name="age" min="1" max="100">

4. Learn <label> for accessibility and placeholder for hints. Connect labels with inputs using for and id.

<label> 
<label> is used to describe an input field
Helps users understand what to enter
Improves accessibility (especially for screen readers)

Example:
<label for="username">Username:</label>
<input type="text" id="username">

placeholder
placeholder gives a hint inside the input box
Disappears when user starts typing

Example:
<input type="text" placeholder="Enter your name">
